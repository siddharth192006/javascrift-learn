1. Introduction to JavaScript

JavaScript is a versatile, lightweight programming language used to make websites dynamic and interactive. It runs in the browser and can manipulate HTML and CSS.

2. Variables and Data Types

Variables store data in JavaScript. Use let and const instead of var for better scoping.

Example Code:

                    
let name = "Siddharth";
let age = 18;
const isStudent = true;

console.log(name, age, isStudent);

3. Operators in JavaScript

JavaScript supports arithmetic, comparison, and logical operators for various operations.

Example Code:

                    
let x = 10, y = 3;
console.log(x + y);  // Addition
console.log(x - y);  // Subtraction
console.log(x * y);  // Multiplication
console.log(x / y);  // Division

4. Functions

Functions are reusable blocks of code. Use arrow functions (=>) for cleaner syntax.

Example Code:

                    
function greet(name) {
    return "Hello, " + name + "!";
}
console.log(greet("Siddharth"));

5. Conditional Statements

Conditional statements allow decision-making in your code.

Example Code:

                    
let age = 18;
if (age >= 18) {
    console.log("You are an adult.");
} else {
    console.log("You are a minor.");
}

6. Loops

Loops allow repeating actions multiple times.

Example Code:

                    
for (let i = 1; i <= 5; i++) {
    console.log(i);
}

7. Arrays

Arrays are lists of items. You can add, remove, or modify their elements.

Example Code:

                    
let fruits = ["Apple", "Banana", "Cherry"];
fruits.push("Mango");  // Add item
console.log(fruits);

8. DOM Manipulation

Use JavaScript to dynamically interact with HTML elements.

Example Code:

                    
document.getElementById("demo").innerHTML = "Hello World!";

10. ES6 Features

ES6 (ECMAScript 2015) introduced modern JavaScript features like arrow functions, template literals, destructuring, and more.

Example Code:

                    
// Arrow Functions
const greet = (name) => `Hello, ${name}!`;

// Template Literals
const message = `Welcome to JavaScript, ${greet("Siddharth")}`;

// Destructuring
const person = { name: "Siddharth", age: 18 };
const { name, age } = person;

console.log(message);
console.log(`Name: ${name}, Age: ${age}`);
